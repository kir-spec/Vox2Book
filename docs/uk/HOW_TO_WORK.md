# Як користуватися Vox2Book

1. Відкрийте теку проєкту в IDE.
2. Покладіть текст у `inputs/raw_texts/`.
3. Скопіюйте промпт з [`prompts/uk/START_USER_PROMPT.md`](../prompts/uk/START_USER_PROMPT.md).
4. Відповідайте на запитання ШІ (або «роби одразу»).
5. Заберіть `.docx` з `output/books/`.

**Промпт:** `prompts/uk/UNIVERSAL_EDITOR_SYSTEM.md`  
**Аудіо → текст (STT):** [`docs/uk/AUDIO_TRANSCRIPTION.md`](AUDIO_TRANSCRIPTION.md) — Whisper, OpenAI API, AssemblyAI, Telegram тощо.  
**8 аудитів:** [`TECHNICAL_SPECIFICATION.md`](TECHNICAL_SPECIFICATION.md)
