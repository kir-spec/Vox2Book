# 📖 Полное руководство пользователя Vox2Book v2.1.0

Добро пожаловать в универсальный издательский комплекс **Vox2Book**!
Программа объединяет **нейросетевой ИИ-модуль вычистки устной речи (LLM)** и **высокоскоростной верстальщик печатных макетов Word (.docx)**.

---

## 🛠️ Способ 1: Оконный графический интерфейс (`vox2book.exe`)
1. Скачайте файл **`vox2book.exe`** из раздела [GitHub Releases](https://github.com/kir-spec/Vox2Book/releases).
2. Запустите файл `vox2book.exe` двойным кликом (приложение работает в собственном графическом окне без консоли).
3. Перетащите ваш файл (расшифровку диктофона `.txt`, `.md` или документ `.docx`) прямо в окно программы.
4. Выберите жанр (Проза, Поэзия, Драматургия, Диалоги) и нажмите кнопку **«🚀 ВЫЧИТАТЬ ТЕКСТ И СФОРМИРОВАТЬ МАКЕТ DOCX»**.

---

## 🤖 Способ 2: Бесплатная локальная Нейросеть (Ollama AI - 100% Оффлайн)
Если вы хотите, чтобы нейросеть вычитывала текст **полностью оффлайн и бесплатно без интернета на вашем ПК**:

1. Установите бесплатный локальный сервер нейросетей [Ollama.com](https://ollama.com).
2. Откройте терминал/консоль и запустите любую русскую модель нейросети:
   ```bash
   ollama run llama3
   # или
   ollama run saiga
   ```
3. Запустите `vox2book.exe` — программа автоматически обнаружит Ollama на `http://localhost:11434` и выполнит глубокое нейросетевое вычитывание вашей речи перед формированием макета DOCX!

---

## ☁️ Способ 3: Облачные Нейросети (ChatGPT / Claude / Gemini Web & API)
Если вы используете веб-версии ChatGPT, Claude или Gemini:

1. Откройте мастер-промпт редактора в репозитории: **[`prompts/MASTER_LLM_PROMPT.md`](file:///E:/coding/%D1%80%D0%B0%D0%B1%D0%BE%D1%82%D0%B0%20%D1%81%20%D0%BB%D0%B8%D1%82%D0%B5%D1%80%D0%B0%D1%82%D1%83%D1%80%D0%BE%D0%B9/prompts/MASTER_LLM_PROMPT.md)**.
2. Скопируйте текст промпта и отправьте его нейросети вместе с вашим сырым аудио-текстом диктофона.
3. Вставьте полученный вычитанный текст нейросети в `vox2book.exe` — приложение мгновенно сделает профессиональное макетирование книги, расставит абзацы, тире, кавычки-ёлочки и сгенерирует печатный файл `.docx`.

---

## ⚡ Способ 4: Автономный быстрой движок Rust (Без Нейросети - 0.1 сек)
Если вам нужна **мгновенная обработка без нейросети**:

1. В окне `vox2book.exe` переключите режим на **«⚡ Автономный движок»**.
2. Программа за 0.1 секунды выполнит:
   - Замену спец-терминов (`"те из бы"` → `USB`, `"в стране джетал"` → `Western Digital`, `"35 размер"` → `3.5"`).
   - Расстановку пропущенных запятых перед союзами (`что`, `чтобы`, `потому что`, `но`, `а`).
   - Книжную типографику (`«...»`, ` — `, частицы `-то`, `-нибудь`, `из-за`).
   - Форматирование стилей Word.

---

## 💻 Способ 5: Режим командной строки (CLI Batch Mode)
Для автоматизации и пакетов файлов:
```powershell
.\vox2book.exe --input "my_transcript.txt" --output "book.docx" --genre prose
```
