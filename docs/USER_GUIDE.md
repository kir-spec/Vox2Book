# 📖 Руководство по работе с Vox2Book API Pipeline v3.0.0

Все файлы результатов **строго сохраняются в папку `output/books/`**! Идея обязательного `.exe` файла упразднена в пользу прямого, чистого и высокоскоростного работы через **API Нейросетей** (`run_api_pipeline.py`).

---

## 📂 Папки проекта

* **`inputs/raw_texts/`** — Сюда кладутся исходные текстовые файлы для вычистки (`.txt`, `.md`, `.docx`).
* **`inputs/audio/`** — Сюда кладутся аудиозаписи для расшифровки (`.ogg`, `.mp3`, `.wav`).
* **`output/books/`** — **Сюда программа СОХРАНЯЕТ готовые вычитанные книги** в формате Word (`.docx`).

---

## 🚀 Как вычитать файл через API:

Запустите команду в терминале:
```bash
python run_api_pipeline.py inputs/raw_texts/ваш_текст.txt
```

Программа вычитает ваш текст через выбранную нейросеть из `config.json` (**LM Studio**, **Ollama**, **OpenAI**, **DeepSeek**, **Claude**) и **сохранит результат в `output/books/ваш_текст.docx`**.
