# 🎭 Цепочка виртуальных корректоров Vox2Book API Pipeline v4.0.0

Проект Vox2Book спроектирован как **автоматизированная цепочка виртуальных корректоров и фильтров** для ИИ-агентов и нейросетей.

Любой входной файл из папки `inputs/raw_texts/` автоматически проходит 5 этапов обработки и сохраняется в папку **`output/books/`**.

---

## ⛓️ 5 Этапов цепочки виртуальных корректоров:

1. **Этап 1: Первичный корректор (STT & Slang Purger)**
   - Вырезает шумы и галлюцинации Whisper STT (`Quiz河`, `"Подписывайтесь на канал"`).
   - Заменяет ошибочный слуховой сленг диктофона (`"те из бы"` → `USB`, `"ссд"` → `SSD-накопитель`, `"в стране джетал"` → `Western Digital`, `"планктом"` → `планктон`).

2. **Этап 2: ИИ-Стилист (Neural LLM Proofreader)**
   - Проводит монолог через выбранную нейросеть (**LM Studio**, **Ollama**, **OpenAI**, **DeepSeek**, **Claude**).
   - Разбивает сплошной текст на смысловые абзацы и литературные предложения, удаляет слова-паразиты.

3. **Этап 3: Издательский корректор (Publisher Typography)**
   - Автоматически расставляет книжные кавычки `«ёлочки»`.
   - Заменяет дефисы на длинные тире ` — `.
   - Форматирует реплики диалогов с префиксом `— `.
   - Исправляет дефисные частицы (`из-за`, `из-под`, `всё-таки`, `в общем`).

4. **Этап 4: Фильтр аудита качества (Quality Auditor)**
   - Проверяет документ на незавершённые предложения или обрывы пунктуации.

5. **Этап 5: Автоматический верстальщик (Book Layout Compiler)**
   - Применяет книжные шрифты (*Times New Roman 12pt*, межстрочный интервал 1.15, красная строка 18pt).
   - **Сохраняет готовый печатный макет в папку `output/books/имя_файла.docx`**.
