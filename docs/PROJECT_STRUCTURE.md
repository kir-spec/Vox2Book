# 📂 Структура проекта и руководство по работе в IDE (Antigravity / Cursor / VS Code)

Это официальное руководство по настройке среды разработки и организации папок проекта **Vox2Book**.

---

## 🛠️ 1. Настройка среды разработки и открытие проекта в IDE

Для корректной работы с проектом и ИИ-ассистентами (Antigravity IDE, Cursor, VS Code, Windsurf):

1. **Откройте вашу IDE** (Antigravity IDE / Cursor / VS Code).
2. Нажмите **`File` → `Open Folder...` (Открыть папку...)**.
3. Укажите корневую папку проекта:
   `E:\coding\работа с литературой`
4. Нажмите **«Выбор папки»**. Теперь IDE с ИИ-агентом работает напрямую с активной рабочей директорией Vox2Book.

---

## 📁 2. Папки и назначение файлов в проекте

```ascii
E:\coding\работа с литературой\
├── inputs/                      # 📥 ВСЕ ВХОДНЫЕ МАТЕРИАЛЫ
│   ├── raw_texts/               # Исходные текстовые файлы (.txt, .md, .docx, .html)
│   └── audio/                   # Сырые аудиозаписи (.ogg, .wav, .mp3, .m4a) для расшифровки
│
├── output/                      # 📤 ГОТОВАЯ ПРОДУКЦИЯ ДЛЯ ПЕЧАТИ
│   └── books/                   # Готовые вычитанные книги в формате Word (.docx)
│
├── prompts/                     # 🧠 ИИ-ПРОМПТЫ И ОБУЧАЮЩИЕ ИНСТРУКЦИИ
│   └── MASTER_LLM_PROMPT.md    # Мастер-промпт главного редактора для нейросетей
│
├── src/                         # ⚙️ ИСХОДНЫЙ КОД ДВИЖКА (RUST)
│   ├── main.rs                  # Точка входа приложения
│   ├── lib.rs                   # Ядро вычистки и генерации
│   ├── llm.rs                   # Клиент API нейросетей (OpenAI, DeepSeek, Claude, Ollama)
│   ├── gui.rs                   # Графический интерфейс приложения
│   └── ...
│
├── config.json                  # 🔑 Файл конфигурации API ключей и провайдеров
├── run_api_pipeline.py          # 🚀 Быстрый запуск обработки через API
├── vox2book.exe                 # 🚀 Исполняемый графический и CLI файл проекта
└── README.md                    # Главная справка проекта
```

---

## 🚀 3. Быстрый регламент работы

1. **Поместите сырые файлы**:
   - Положите файлы с текстом или расшифровками устной речи в папку: `inputs/raw_texts/`.
   - Положите аудиозаписи для расшифровки в папку: `inputs/audio/`.

2. **Настройте API (если нужно)**:
   - Введите ваш ключ API в файл `config.json` (OpenAI, DeepSeek или Anthropic Claude).

3. **Запустите обработку**:
   - Через окно приложения: запустите `vox2book.exe` двойным кликом.
   - Или через скрипт API в консоли IDE:
     ```bash
     python run_api_pipeline.py
     ```

4. **Заберите готовую книгу**:
   - Готовая вычитанная книга в формате DOCX сохранится в папку: `output/books/`.
