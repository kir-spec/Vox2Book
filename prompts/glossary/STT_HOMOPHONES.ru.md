# STT / OCR — таблица кандидатов (русский)

> **Не загружается автоматически.**  
> **Не использовать как скрипт замены.** Только контекст — см. [`CONTEXTUAL_TYPO_CORRECTION_GUIDE.ru.md`](CONTEXTUAL_TYPO_CORRECTION_GUIDE.ru.md).

Другие языки: [EN](STT_HOMOPHONES.en.md) · [UK](STT_HOMOPHONES.uk.md) · [индекс](STT_HOMOPHONES.example.md)

---

## Формат записи в JSON

```json
{
  "heard": "юсб|те из бы",
  "correct": "USB",
  "context_hint": "кабель, порт, флешка",
  "never_replace_globally": true
}
```

## Разговорная речь (обычно НЕ «исправлять» в литературное)

| Услышано | Действие |
|----------|----------|
| ну, э-э, мм, а-а | Умеренно сократить |
| короче, типа, как бы | Сохранить в живом диалоге |
| щас, чё, норм, ок | Допустимо в диалоге |
| блин, ё-моё, чёрт | **Не трогать** без просьбы |

## Омофоны (только по смыслу фразы)

| Пара | Различие по контексту |
|------|------------------------|
| ключ | дверь vs музыкальный |
| лук | овощ vs оружие |
| коса | причёска vs инструмент |
| плата | деньги vs печатная плата (IT) |
| мышь | компьютер vs животное |
| что бы / чтобы | союз цели vs «что бы ты ни сказал» |
| не / ни | отрицание vs «ни разу» |

## Галлюцинации Whisper — удалять всегда

- «Подписывайтесь на канал», «Спасибо за просмотр», «Продолжение следует…»
- «Редактор субтитров», «Субтитры сделал»
- «DimaTorzok», «Quiz河», повтор фразы 3+ раз подряд
