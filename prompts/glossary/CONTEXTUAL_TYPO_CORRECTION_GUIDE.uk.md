## Залізне правило (не порушувати)

> **Виправляй слова й фрази ЛИШЕ в контексті — ніколи окремо, як автокоректор.**

| Заборонено | Чому |
|------------|------|
| «Лікувати» кожне підозріле слово зі словника | Омофони й STT часто **граматично правильні**, але **семантично хибні** |
| Глобальна заміна `X → Y` по всьому файлу | Те саме «почуте» слово в різних місцях означає різне |
| Довіряти спелчекеру без контексту | Він ранжує орфографію, а не **зміст репліки** |
| Вгадувати мовчки, якщо контекст не сходиться | Краще залишити джерело й запитати |

**Мінімальний контекст:**
1. **ціле речення** (краще — абзац або пара сусідніх реплік);
2. **тема** (побут, здоров'я, IT, стосунки…);
3. **хто говорить** (ім'я спікера);
4. **глосарій проєкту** (`config/glossary_user.json`).

Якщо лишається **2+ правдоподібних читання** — **не прав**; у звіт або запитай.

## Алгоритм (для ШІ)

```
1. Прочитай абзац / репліку цілком — зрозумій намір.
2. Познач аномалії: безглуздість, зламані колокації, неузгодженість, чужий жаргон.
3. Сформулюй 2–5 кандидатів — цілу фразу, не одне слово з таблиці.
4. Обери варіант, де:
   - речення звучить природно усно;
   - узгоджені відмінки, час, особа;
   - сусідня репліка не ламається (питання ↔ відповідь).
5. Впевненість < 80% — залиш джерело + [?] або питання користувачу.
```

**Приклад (контекст вирішує):**

| Джерело STT | Погано (без контексту) | Добре (в контексті) |
|-------------|------------------------|---------------------|
| «у мене температура валяюсь» | механічна заміна слів | зберегти зміст: хвороба, відпочинок |
| «треба купити мишу» | тварина | комп'ютерна миша — якщо про ПК |
| «залив на прод» | продуктовий магазин | **production**, якщо IT-діалог |

## Типи помилок

### 1. Омофони та слухові підстановки (STT)
Звучать схоже: *мир / мир*, *ключ / ключ* (різний зміст).

### 2. Разом / окремо через паузи
*«взагалі»*, *«нічого»* — інколи стиль, інколи помилка.

### 3. Імена, бренди, топоніми
Whisper ламає рідкісні імена — **лише з глосарію користувача**.

### 4. Code-switching і IT-жаргон
Українська + англійські терміни; кирилична «фонетика» (*кафка → Kafka*).

### 5. Галюцинації Whisper
«Підписуйтесь на канал», субтитри, сміття іншою мовою — **видаляти**.

### 6. OCR / опечатки клавіатури
*і/ї*, *е/и*, перестановки літер — якщо слово **не в темі**.

### 7. Омографи
Однакове написання — різний зміст: *замок*, *коса*, *лук* — **лише за сусідніми словами**.

## Відкриті ресурси (довідник, не сліпа заміна)

| Ресурс | Що дає | Посилання |
|--------|--------|-----------|
| **Kartaslov orfo_and_typos** | Частотні опечатки (корисно для UA+RU mixed) | https://github.com/dkulagin/kartaslov/tree/master/dataset/orfo_and_typos |
| **Silero homographs** | Омографи (багато спільного з UA) | https://github.com/snakers4/silero-stress/wiki/Homograph-List |
| **JamSpell** | Контекстний спелчекер; модель `uk` | https://github.com/bakwc/JamSpell |
| **ai-forever/spellcheck_benchmark** | Пари речень RU (корисно для східнослов'янського) | https://huggingface.co/datasets/ai-forever/spellcheck_benchmark |
| **Yandex.Speller** | API; перевіряй зміст вручну | https://yandex.ru/dev/speller/ |
| **LanguageTool** | UA граматика (допоміжно) | https://languagetool.org/ |

**Висновок для ШІ:** навіть JamSpell помиляється на омофонах і жаргоні. Ти — **редактор із розумінням змісту**.

## Часті STT-пари (кандидати — не автозаміна!)

Детальні таблиці: [`STT_HOMOPHONES.uk.md`](STT_HOMOPHONES.uk.md)

### Побут, здоров'я

| Почуто | Ймовірно | Коли вірно |
|--------|----------|------------|
| валяюсь / валясь | валяюсь | хвороба, втома, дім |
| температура | температура (тіло) | самопочуття, не погода |
| роз'єм | роз'єм | зарядка, кабель |
| тиск скаче | артеріальний тиск | здоров'я |

### IT (скорочено)

| Почуто | Ймовірно | Контекст |
|--------|----------|----------|
| юсб | USB | кабель, порт |
| прод / на прод | production | деплой, не «продуктовий» |
| кафка | Kafka | бекенд, черги |
| гіт / гітхаб | Git / GitHub | код |
| телеграм / телега | Telegram | месенджер |

### Розмовна мова (зазвичай зберегти)

| Почуто | Примітка |
|--------|----------|
| ну, е-е, типу, коротше | помірно скоротити |
| зараз, норм, ок | допустимо в діалозі |
| лайка | **не прибирати** без прохання |

### Омофони (лише за змістом фрази)

| Пара | Різниця |
|------|---------|
| ключ | двері vs музичний |
| лук | овоч vs зброя |
| мир | спокій vs світ |
| не / ні | заперечення vs «ні разу» |
| що б / щоб | мета vs «що б не сталося» |

## Матриця схожих літер (OCR)

Часто плутають **і/ї**, **е/и**, **в/у**, **п/р**, **т/ь**. Перевір, чи слово **в цій темі** існує.

---

## Антипатерни

1. **`прод` → продуктовий** в IT-чаті — ламає зміст.
2. **`миша` → тварина** в тексті про комп'ютер.
3. **Прибрати сленг/лайку** без прохання.
4. **Імена з чужих прикладів** глосарію.
5. **Глобальна заміна** по файлу.
