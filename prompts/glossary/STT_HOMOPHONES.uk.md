## Формат запису в JSON

```json
{
  "heard": "юсб|у ес бі",
  "correct": "USB",
  "context_hint": "кабель, порт, флешка",
  "never_replace_globally": true
}
```

## Розмовна мова (зазвичай НЕ «виправляти»)

| Почуто | Дія |
|--------|-----|
| ну, е-е, мм | Помірно скоротити |
| коротше, типу, як би | Зберегти в живому діалозі |
| зараз, норм, ок | Допустимо в діалозі |
| лайка, евфемізми | **Не чіпати** без прохання |

---

## Омофони та плутанина (лише за змістом)

| Пара | Різниця за контекстом |
|------|------------------------|
| ключ | двері vs музичний |
| лук | овоч vs зброя |
| мир | спокій vs світ (планета) |
| не / ні | заперечення vs «ні разу» |
| що б / щоб | сполучник мети vs «що б ти не сказав» |
| і / й / ї | перевіряти за правилами UA, не механічно |
| в / у | прийменники — за нормою, не всюди «в» |

## Галюцинації Whisper — видаляти завжди

- «Підписуйтесь на канал», «Дякуємо за перегляд»
- «Продовження слідує…», «Редактор субтитрів»
- Повтор однієї фрази 3+ рази підряд
- Сміття іншою мовою, не пов'язане з мовцем
